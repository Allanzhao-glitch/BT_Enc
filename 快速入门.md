```python
import asyncio
from bluetooth_secure_client import BluetoothSecureClient

async def main():
```

### 完整示例

```python
import asyncio
from bluetooth_secure_client import BluetoothSecureClient

async def secure_communication():
    # 创建安全客户端
    client = BluetoothSecureClient("YourDeviceName")
    
    try:
        # 1. 搜索设备
        print("搜索设备...")
        if not await client.search_device(timeout=10.0):
            print("设备未找到")
            return
        
        # 2. 连接设备
        print("连接设备...")
        if not await client.connect():
            print("连接失败")
            return
        
        # 3. 执行密钥交换握手
        print("执行握手...")
        if not await client.perform_handshake():
            print("握手失败")
            return
        
        print("✓ 安全连接已建立!")
        
        # 4. 发送加密消息
        message = b"Hello Secure World!"
        await client.send_encrypted_data(message)
        print(f"✓ 已发送: {message.decode()}")
        
        # 5. 接收加密响应 (可选)
        try:
            response = await client.receive_encrypted_data(timeout=5.0)
            print(f"✓ 已接收: {response.hex()}")
        except asyncio.TimeoutError:
            print("等待响应超时")
        
    except Exception as e:
        print(f"✗ 错误: {e}")
    
    finally:
        # 6. 断开连接
        await client.disconnect()

# 运行
asyncio.run(secure_communication())
```

## 📊 核心API

### BluetoothSecureClient

| 方法 | 说明 | 返回值 |
|-----|------|--------|
| `search_device(timeout)` | 搜索设备 | bool |
| `connect()` | 连接设备 | bool |
| `perform_handshake()` | 执行握手 | bool |
| `send_encrypted_data(data)` | 发送加密数据 | bool |
| `receive_encrypted_data(timeout)` | 接收加密数据 | bytes |
| `disconnect()` | 断开连接 | None |

### EncryptionProtocol

| 方法 | 说明 | 返回值 |
|-----|------|--------|
| `generate_ecdh_keypair()` | 生成密钥对 | (公钥, 私钥) |
| `compute_shared_secret(peer_pub)` | 计算共享密钥 | bytes |
| `aes256_cbc_encrypt(plaintext)` | CBC加密 | bytes |
| `aes256_cbc_decrypt(ciphertext)` | CBC解密 | bytes |
| `aes256_gcm_encrypt(plaintext)` | GCM加密 | (密文, 标签) |
| `aes256_gcm_decrypt(ciphertext, tag)` | GCM解密 | bytes |

## 🐛 常见问题

### Q1: 提示 "ModuleNotFoundError: No module named 'cryptography'"

**解决**: 安装依赖包
```bash
python -m pip install cryptography bleak
```

### Q2: 搜索不到设备

**检查**:
1. 蓝牙是否开启
2. 设备名称是否正确
3. 设备是否在广播模式
4. 增加搜索超时时间

```python
await client.search_device(timeout=20.0)  # 增加到20秒
```

### Q3: 连接失败

**检查**:
1. UUID配置是否正确
2. 设备是否已被其他程序连接
3. 蓝牙权限是否足够

### Q4: 握手失败

**检查**:
1. 静态密钥配置是否与服务器一致
2. 数据包格式是否正确
3. 查看日志中的详细错误信息

### Q5: 解密失败

**检查**:
1. 共享密钥是否计算正确
2. GCM标签是否完整
3. AAD数据是否一致

## 🎯 实战场景

### 场景1: 物联网设备控制

```python
async def control_iot_device():
    async with BluetoothSecureClient("IoT-Device-001") as client:
        if await client.search_device():
```

### 场景2: 数据采集

```python
async def collect_sensor_data():
    async with BluetoothSecureClient("Sensor-Device") as client:
        if await client.search_device():
            if await client.connect():
                if await client.perform_handshake():
                    # 请求数据
```

### 场景3: 固件更新

```python
async def firmware_update():
    async with BluetoothSecureClient("Device-Update") as client:
        if await client.search_device():
            if await client.connect():
                if await client.perform_handshake():
                    # 读取固件文件
                    with open("firmware.bin", "rb") as f:
                        firmware_data = f.read()
                    
                    # 分块发送
                    chunk_size = 128
                    for i in range(0, len(firmware_data), chunk_size):
                        chunk = firmware_data[i:i+chunk_size]
                        await client.send_encrypted_data(chunk)
                        await asyncio.sleep(0.1)
```
